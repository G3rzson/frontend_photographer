# 📸 Photographer Portfolio Website

A modern and responsive portfolio website for photographers built with Next.js, TypeScript, and Tailwind CSS. This project demonstrates my skills in building frontend web applications with modern technologies.

> 🎨 **Note**: Gallery images are AI-generated demo content created with Google Gemini and Canva to showcase the website's functionality.

## ✨ Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- 🌓 Dark/Light mode toggle
- 🖼️ Image gallery with collapsible categories and carousel
- 💰 Pricing page with service packages
- 📧 Contact form with validation (React Hook Form + Zod)
- ⚡ Optimized performance with Next.js Image component
- ✅ Unit tests with Vitest and React Testing Library

## 🛠️ Tech Stack

- **Framework**: Next.js 16.2 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **UI Components**: Shadcn/ui
- **Form Validation**: React Hook Form + Zod
- **Testing**: Vitest + React Testing Library
- **Icons**: Lucide React
